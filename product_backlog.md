# Product Backlog: Bug Predictor

| ID | Title | Description | Priority |
|----|-------|-------------|----------|
| US-01 | **Analyze Git History** | As a user, I want to analyze the git history to find frequently fixed files so I can identify unstable code. | High |
| US-02 | **Calculate Complexity** | As a developer, I want to calculate the cyclomatic complexity of files to assess maintainability. | High |
| US-03 | **Calculate Risk Score** | As a user, I want a single risk score per file combining churn and complexity. | High |
| US-04 | **View Dashboard** | As a user, I want a web interface to view the list of at-risk files sorted by score. | Medium |
| US-05 | **Export Report** | As a user, I want to export the analysis results. | Low |
